# Ubuntu User Management Guide

## 1. Creating a New User in One Command

To create a new user with a home directory, default shell (`/bin/bash`), and a predefined password:

```bash

```

- `sudo` â†’ Runs the command with administrative privileges.
- `useradd` â†’ Creates a new user.
- `-m` â†’ Automatically creates a home directory (`/home/username`).

- `-p $(openssl passwd -1 "YourPassword")` â†’ Encrypts the password before setting it (MD5-based encryption).

## 2. Adding a User to the Docker Group

If you want the user to run Docker commands without `sudo`, add them to the `docker` group:

```bash
sudo usermod -aG docker username
```

### **Explanation:**
- `sudo` â†’ Runs the command with administrative privileges.
- `usermod` â†’ Modifies user properties.
- `-aG docker` â†’ Appends (`-a`) the user to the `docker` group (`-G` stands for group).
- `username` â†’ Replace this with the actual username.

## 3. Deleting an Existing User with All Resources

To remove a user along with their home directory, mail spool, and other related files:

```bash
sudo userdel -r username
```

### **Explanation:**
- `sudo` â†’ Runs the command with administrative privileges.
- `userdel` â†’ Deletes a user.
- `-r` â†’ Removes the home directory and mail spool (`/var/mail/username`).
- `username` â†’ Replace with the actual username.
